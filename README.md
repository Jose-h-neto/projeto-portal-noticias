# Portal de Notícias

Projeto front-end desenvolvido com HTML e CSS com o objetivo de praticar fundamentos de desenvolvimento web e construção de layouts.

## Visão geral

Este projeto consiste em um portal de notícias estático, com foco na organização de conteúdo, estrutura semântica e separação de estilos.

## Objetivos

- Praticar HTML semântico  
- Trabalhar com layouts utilizando CSS  
- Organizar arquivos de forma modular  
- Utilizar imagens, ícones e fontes externas  
- Aplicar boas práticas de separação de estilos  

## Estrutura do projeto

```
PROJETO-PORTAL-NOTICIAS
├── assets/
│ ├── icons/

├── styles/

│ ├── header.css
│ ├── index.css
│ ├── sections.css
│ └── utility.css

```

## Funcionalidades

- Header com navegação principal e secundária  
- Seção de notícias em destaque  
- Lista de artigos mais lidos  
- Área temática (ex: Inteligência Artificial)  
- Sidebar com conteúdos adicionais e anúncios  
- Footer simples  

## Tecnologias utilizadas

- HTML5  
- CSS3  
- Google Fonts (Archivo)  

## Organização do CSS

O projeto utiliza separação de estilos por responsabilidade:

- `global.css`: estilos globais e reset  
- `header.css`: estilos do cabeçalho  
- `sections.css`: estilos das seções principais  
- `utility.css`: classes utilitárias  
- `index.css`: estilos específicos da página principal  

## Como executar

1. Abra o arquivo `index.html` no navegador  

ou  

2. Utilize um servidor local (recomendado)  
   - Ex: extensão Live Server no VS Code
